About
Dear visitors,

I am excited to introduce you to this invented web service, which is my pet project aimed at enhancing my backend web development skills using Python and Django.
The database is based on three interconnected models: Newspapers, Topics, and Redactors. Each topic can have several newspapers, each newspaper can have several redactors, and each redactor can work on several newspapers. On this website, you can create, modify, and delete any of these data using web-interface.

In addition, there are implemented user authentication with secure creation of new users. Passwords in the database are stored in a hashed form, ensuring the safety of our users' personal information.
